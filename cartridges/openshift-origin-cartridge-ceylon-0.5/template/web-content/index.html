Welcome to Ceylon on RedHat's OpenShift cloud

Write your own Ceylon application and replace this sample.

In order to commit to your new project, go to your projects git repo directory.

For example, if you named your application myfirstapp, you would find dynamic contents of this app located under myfirstapp/sources and static files under myfirstapp/web-content. You can edit that and push your changes from the myfirstapp directory by running: 

    git commit -a -m 'Some commit message'
    git push

Then reload this page.

OpenShift is configured to execute toplevel method "run" in a "ceylon.demo.net" module. You can change module name in "ceylon.properties" file located in "myfirstapp/.openshift/config/".

When you push Ceylon sources, they are compiled by OpenShif, if you want to upload pre compiled modules, you can place them in "myfirstapp/.openshift/modules/" folder.
